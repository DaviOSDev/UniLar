# Imagens do UniLar

Coloque aqui imagens usadas no projeto (logo, ícones, ilustrações).

## Uso no código

Importe as imagens nos componentes:

```tsx
import logo from '@/assets/images/logo.png';

<img src={logo} alt="UniLar" className="h-10" />
```

Ou com caminho relativo a partir de `src`:

```tsx
import logo from '../assets/images/logo.png';
```

## Arquivos

- `logo.png` — Logo do UniLar (casa em vermelho e azul)
